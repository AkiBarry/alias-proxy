# alias-proxy
Allows you to make redirects in Windows.

Comes with a mapping.json with some values already filled in.
```json
{
  "jupyter": "127.0.0.1:8888",
  "goog": "google.com"
}
```

This messes with your hosts file. Use at your own risk.

## Requirements
Docker
Python

## How to run
Run `setup.bat`
